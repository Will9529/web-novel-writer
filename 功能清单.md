### 质检中心功能详情

| 检查项 | 说明 | 扣分 |
|--------|------|------|
| 禁用词检测 | 检测高频烂词（突然、瞬间、顿时等） | -20 |
| 结构相似度 | 与最近 5 章对比，防止重复 | -15 |
| 情节债务核查 | 检查应填未填的伏笔 | -20 |
| 字数检查 | 2500-3000 字硬性校验 | -10/-5 |
| 格式检查 | 段落过长、连续对话过多 | -5 |

**评分标准：**
- 90-100 分：优秀 ✅
- 70-89 分：良好 ⚠️
- <70 分：需修改 ❌

**修改建议：** AI 生成针对性修改建议

## 🚀 快速开始

### 1. 安装依赖
```bash
cd web_novel_writer
pip install -r requirements.txt
```

### 2. 配置 API Key
编辑 `config.yaml`，填入阿里云通义 API 密钥：
```yaml
aliyun:
  api_key: "sk-你的 API Key"
  model: "qwen-plus"
```

### 3. 启动应用
```bash
python main.py
```

或双击 `start.bat`（Windows）

## ⚙️ 配置说明

### config.yaml 完整配置

```yaml
# 阿里云 API 配置
aliyun:
  api_key: "sk-xxxxxxxx"      # 必填
  model: "qwen-plus"          # qwen-turbo/plus/max

# 项目设置
project:
  default_chapter_words: 2750  # 目标字数
  quality_check_enabled: true  # 启用质检
  auto_save: true             # 自动保存

# 禁用词表
banned_words:
  - 突然
  - 瞬间
  - 顿时
  - 不禁
  - 只见
```
