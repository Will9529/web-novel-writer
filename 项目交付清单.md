## 🚀 Windows 使用说明

### 方式一：源码运行
```
1. 安装 Python 3.8+（勾选 Add to PATH）
2. 双击 install.bat
3. 编辑 config.yaml 填写 API Key
4. 双击 start.bat
```

### 方式二：打包成 EXE
```
1. 完成方式一的步骤 1-3
2. 双击 build.bat
3. 从 dist/ 获取 exe 文件
4. 将 config.yaml 复制到 exe 同目录
```
