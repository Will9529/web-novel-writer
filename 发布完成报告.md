## 🛠️ 问题排查记录

### 遇到的问题

#### 1. GitHub 推送失败（网络问题）
**现象**: `GnuTLS recv error (-110): The TLS connection was non-properly terminated`

**原因**: 服务器到 GitHub 的网络连接不稳定

**解决**: 使用带 token 的 URL 进行认证推送
```bash
git remote set-url origin https://TOKEN@github.com/USER/REPO.git
```

#### 2. Git 仓库损坏
**现象**: `fatal: did not receive expected object`

**原因**: 之前的 git 仓库状态异常

**解决**: 重新初始化 git 仓库
```bash
rm -rf .git
git init
git add .
git commit -m "初始提交"
git push -u origin main --force
```

#### 3. 缺少 zip 命令
**现象**: `/usr/bin/bash: line 3: zip: command not found`

**解决**: 使用 Python 的 `shutil.make_archive()` 创建 ZIP 包

## 🚀 使用指南

### 方式一：下载 ZIP 包

1. **下载**: https://github.com/Will9529/web-novel-writer/releases/download/v1.0.0/web_novel_writer_v1.0.zip

2. **解压**到本地

3. **测试运行**:
   - Windows: 双击 `run_test.bat`

4. **配置 API Key**:
   - 编辑 `config.yaml`
   - 填入阿里云 API Key

5. **打包 EXE** (可选):
   - Windows: 双击 `build.bat`

### 方式二：克隆仓库

```bash
git clone https://github.com/Will9529/web-novel-writer.git
cd web-novel-writer

# 安装依赖
pip install -r requirements.txt

# 运行应用
python main.py
```
