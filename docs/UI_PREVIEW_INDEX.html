如何保存：右键点击 SVG 图片 → 另存为 → 保存为 .svg 文件，可用 Illustrator/Figma 编辑。

如何转 PNG：用浏览器打开后截图，或使用在线工具 (如 cloudconvert.com) 转换 SVG 到 PNG。
