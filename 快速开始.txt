========================================
  网文 AI 辅写系统 - Windows 快速开始
========================================

【第一步】安装 Python
----------------------
1. 访问 https://www.python.org/downloads/
2. 下载 Python 3.10 或更高版本
3. 运行安装程序
4. ⚠️ 重要：勾选 "Add Python to PATH"
5. 点击 "Install Now"


【第二步】获取阿里云 API Key
----------------------
1. 访问 https://dashscope.console.aliyun.com/
2. 登录阿里云账号（没有就注册一个）
3. 点击左侧 "API-KEY 管理"
4. 点击 "创建新的 API-KEY"
5. 复制 Key（格式类似：sk-xxxxxxxxxxxx）


【第三步】配置项目
----------------------
1. 用记事本打开 config.yaml
2. 找到这一行：api_key: ""
3. 改成：api_key: "sk-你的 Key"
4. 保存文件


【第四步】运行安装脚本
----------------------
1. 打开项目文件夹
2. 双击 install.bat
3. 等待依赖安装完成（约 1-2 分钟）
4. 看到"安装完成！"提示


【第五步】启动应用
----------------------
1. 双击 start.bat
2. 应用窗口弹出
3. 点击"新建项目"选择保存位置
4. 开始创作！


========================================
  界面功能说明
========================================

📜 风格宪法标签页
----------------------
- 手动编写你的写作风格规范
- 或点击"从样章自动提炼风格"上传你的样章
- 点击"保存风格宪法"保存


📝 章纲编写标签页
----------------------
- 输入章节号（如第 1 章）
- 填写六要素：场景、人物、事件、开场、结尾、承上启下
- 点击"AI 辅助填充"让 AI 帮你生成初稿
- 点击"保存章纲"保存


🌍 世界状态标签页
----------------------
- 追踪人物状态、情节债务、已用模板
- 点击"加载最新状态"查看当前状态
- 点击"生成状态变更 Diff"自动对比变化


========================================
  常见问题
========================================

Q: 双击 bat 文件没反应？
A: 右键 → 以管理员身份运行

Q: 安装时出现红色错误？
A: 检查网络连接，或手动执行：
   pip install PyQt6 qianfan PyYAML markdown requests

Q: 启动后提示 API Key 错误？
A: 确认 config.yaml 中 api_key 已正确填写

Q: 想要打包成 exe 发给别人？
A: 双击 build.bat，生成的 exe 在 dist 文件夹
