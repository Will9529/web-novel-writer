## 🚀 使用流程

### 开发者（有 Python 环境）

```bash
# 1. 克隆/下载项目
git clone https://github.com/Will9529/web-novel-writer.git
cd web-novel-writer

# 2. 安装依赖
pip install -r requirements.txt

# 3. 配置 API Key
# 编辑 config.yaml，填入阿里云 API Key

# 4. 运行应用
python main.py
```

### 普通用户（无 Python 环境）

```bash
# 1. 双击运行 build.bat
# 等待打包完成（约 1-3 分钟）

# 2. 进入 dist/网文 AI 辅写系统/
# 双击 网文 AI 辅写系统.exe

# 3. 配置 API Key
# 编辑 config.yaml，填入阿里云 API Key
```

## ⚠️ 注意事项

### 1. API Key 配置
- 必须填写阿里云通义 API Key 才能使用 AI 功能
- 获取地址：https://dashscope.console.aliyun.com/
- 推荐使用 `qwen-plus` 模型（平衡性能和成本）

### 2. 网络要求
- 需要网络连接访问阿里云 API
- 防火墙需放行 `dashscope.aliyuncs.com`

### 3. 打包说明
- 首次打包需要下载 PyInstaller（约 50MB）
- 打包后文件大小约 100-150MB
- 打包时间：1-3 分钟（取决于电脑性能）

### 4. GitHub 推送
- 本地提交已完成
- GitHub 仓库已创建：https://github.com/Will9529/web-novel-writer
- 因网络问题，推送需手动执行：
  ```bash
  cd web_novel_writer
  git push -u origin main
  ```

2. 查看文档：`使用说明完整版.md`
3. 测试指南：`测试运行指南.md`

### 常见问题
- **应用无法启动**: 检查 Python 版本（需 3.8+）
- **API 错误**: 检查 API Key 是否正确
- **界面模糊**: 右键.exe → 属性 → 兼容性 → 高 DPI 设置
- **打包失败**: 确保已安装所有依赖
