## 一、游戏概述

### 一句话介绍
**怪物入侵，玩家只有一辆车，收集物资升级车辆管理人员，预防怪物的同时需要注意车上的水/食物/能源**

### 核心体验
```
🚗 开车逃亡 → 📦 收集物资 → 🔧 升级车辆 → 👥 管理幸存者 → 🔫 防御怪物 → 🔄 循环
```

### 参考游戏
| 游戏 | 借鉴元素 |
|------|----------|
| Frostpunk | 资源管理、道德选择 |
| 死亡搁浅 | 运输、路线规划 |
| 疯狂麦克斯 | 车辆改装、末世美学 |
| This War of Mine | 幸存者管理、道德困境 |

## 二、核心玩法循环

```
┌─────────────────────────────────────────────────────────┐
│                    游戏循环                              │
│                                                         │
│   ┌──────────┐    ┌──────────┐    ┌──────────┐        │
│   │  路上    │ →  │  收集    │ →  │  管理    │        │
│   │  行驶    │    │  物资    │    │  车辆    │        │
│   └──────────┘    └──────────┘    └──────────┘        │
│        ↑                                      │        │
│        │                                      ↓        │
│   ┌──────────┐                        ┌──────────┐    │
│   │  继续    │ ←─────────────────────  │  防御    │    │
│   │  前进    │                        │  袭击    │    │
│   └──────────┘                        └──────────┘    │
└─────────────────────────────────────────────────────────┘
```

### 单局流程
| 阶段 | 时长 | 玩法 |
|------|------|------|
| 出发准备 | 2 分钟 | 检查资源、分配人员、选择路线 |
| 路上行驶 | 5-10 分钟 | 随机事件、资源点选择 |
| 收集物资 | 3-5 分钟 | 探索、战斗、决策 |
| 营地休整 | 3 分钟 | 升级车辆、分配工作、消耗资源 |
| 夜间防御 | 2-3 分钟 | 塔防式战斗 |
| 循环 | - | 继续前进或返回 |

## 三、核心系统

### 3.1 资源系统

| 资源 | 用途 | 获取方式 | 消耗速率 |
|------|------|----------|----------|
| **燃料** | 车辆行驶 | 加油站、油桶 | 10/公里 |
| **食物** | 幸存者生存 | 超市、仓库 | 1 人/天 |
| **水** | 幸存者生存 | 水井、河流 | 1 人/天 |
| **弹药** | 防御战斗 | 警局、军械库 | 战斗消耗 |
| **零件** | 车辆维修 | 修车厂、拆解 | 维修消耗 |
| **材料** | 车辆升级 | 工厂、仓库 | 升级消耗 |

### 3.2 车辆系统

```
车辆属性
├── 生命值 (HP) - 归零游戏结束
├── 燃料箱 - 决定续航
├── 货仓容量 - 决定携带资源上限
├── 装甲值 - 减少受伤
└── 速度 - 影响行驶时间和逃跑率
```

**可升级部件**:
| 部件 | 等级 | 效果 | 材料消耗 |
|------|------|------|----------|
| 引擎 | 1-5 级 | +速度，+燃料消耗 | 零件×5/级 |
| 装甲 | 1-5 级 | +生命，-速度 | 材料×10/级 |
| 货仓 | 1-5 级 | +容量 | 材料×8/级 |
| 武器 | 1-3 级 | +防御伤害 | 零件×10 + 弹药×20/级 |
| 水箱 | 1-3 级 | +水容量 | 材料×5/级 |

### 3.3 幸存者系统

**幸存者属性**:
| 属性 | 说明 |
|------|------|
| 生命 | 归零死亡 |
| 心情 | 过低会叛变或自杀 |
| 技能 | 驾驶/维修/战斗/医疗 |
| 状态 | 正常/受伤/生病 |

**工作分配**:
| 岗位 | 需要技能 | 效果 |
|------|----------|------|
| 驾驶员 | 驾驶 | 减少燃料消耗 |
| 维修工 | 维修 | 自动修复车辆 |
| 守卫 | 战斗 | 增加防御伤害 |
| 医生 | 医疗 | 治疗伤员 |

### 3.4 地图系统

```
地图结构：线性关卡 + 分支路线

起点 → [分支 A/B/C] → [分支 A/B] → 终点 (安全区)
```

**地点类型**:
| 类型 | 资源 | 风险 |
|------|------|------|
| 废弃加油站 | 燃料×50 | 低 |
| 超市 | 食物×30, 水×30 | 低 |
| 军营 | 弹药×50 | 高 |
| 医院 | 医疗包×10 | 中 |
| 工厂 | 材料×40, 零件×20 | 中 |
| 幸存者营地 | 招募幸存者 | 低 |
| 怪物巢穴 | 大量资源 | 极高 |

### 3.5 战斗系统

**夜间防御** (简化塔防):
```
怪物波次 → 选择防御策略 → 自动战斗 → 结算
```

**防御策略**:
| 策略 | 弹药消耗 | 伤害 | 风险 |
|------|----------|------|------|
| 保守防御 | -10 | 低 | 车辆可能受伤 |
| 正常防御 | -20 | 中 | 平衡 |
| 激进防御 | -40 | 高 | 可能浪费弹药 |

## 四、数值设计

### 4.1 基础数值

```
# 幸存者消耗
每人每天消耗：食物 1, 水 1
每人基础心情：50/100

# 车辆基础
初始生命值：100
初始燃料容量：100
初始货仓容量：200
初始速度：50 km/h

# 资源上限
食物上限：200
水上限：200
燃料上限：100
弹药上限：100
```

### 4.2 距离与天数

```
总游戏天数：15 天
安全区距离：500 公里
平均每天行驶：35-40 公里

事件触发频率：
- 小事件：每 5 公里
- 资源点：每 15 公里
- 怪物袭击：每 3 天夜间
```

### 4.3 难度曲线

| 天数 | 怪物强度 | 资源稀缺度 | 事件复杂度 |
|------|----------|------------|------------|
| 1-3 天 | ⭐ | ⭐ | 简单 |
| 4-7 天 | ⭐⭐ | ⭐⭐ | 中等 |
| 8-11 天 | ⭐⭐⭐ | ⭐⭐⭐ | 困难 |
| 12-15 天 | ⭐⭐⭐⭐ | ⭐⭐⭐⭐ | 极难 |

## 五、事件系统

### 5.1 随机事件池

| 事件 | 描述 | 选择 |
|------|------|------|
| 抛锚 | 车辆故障 | 修 (耗零件)/推车 (耗时间) |
| 迷雾 | 能见度低 | 慢行 (耗燃料)/快速 (危险) |
| 路障 | 道路阻塞 | 清理 (耗时间)/绕路 (耗燃料) |
| 求救 | 幸存者求救 | 救 (耗资源)/无视 (扣心情) |
| 商人 | 流动商人 | 交易/抢劫 |
| 陷阱 | 敌人陷阱 | 拆除 (危险)/绕行 |

### 5.2 道德选择事件

| 事件 | 选择 A | 选择 B |
|------|--------|--------|
| 受伤幸存者 | 救 (耗医疗包 +1 人) | 不救 (保持资源) |
| 陌生车队 | 分享资源 (建立关系) | 拒绝 (可能被抢) |
| 危险任务 | 接受 (高风险高回报) | 拒绝 (安全) |

## 六、胜利与失败

### 胜利条件
```
到达安全区 (500 公里)
至少 1 名幸存者存活
车辆耐久 > 0
```

### 失败条件
```
车辆生命值归零
所有幸存者死亡
资源耗尽无法前进
```
