# 🖋️ 网文 AI 辅写系统 - 测试运行指南

## 📋 测试前准备

### 1. 系统要求

| 项目 | 要求 |
|------|------|
| 操作系统 | Windows 10/11 |
| Python 版本 | 3.8 或更高 |
| 网络连接 | 需要访问阿里云 API |
| 内存 | 至少 2GB 可用 |

### 2. 获取 API Key

1. 访问：https://dashscope.console.aliyun.com/
2. 登录阿里云账号
3. 进入 **API-KEY 管理**
4. 创建新 Key
5. 复制 Key 到配置文件

---

## 🚀 运行步骤

### 方式一：一键启动（推荐）

双击运行 `run_test.bat`

脚本会自动：
- 检查 Python 环境
- 创建虚拟环境
- 安装依赖
- 启动应用

### 方式二：手动运行

```bash
# 1. 打开命令行，进入项目目录
cd web_novel_writer

# 2. 创建虚拟环境
python -m venv venv

# 3. 激活虚拟环境
venv\Scripts\activate

# 4. 安装依赖
pip install -r requirements.txt

# 5. 配置 API Key（编辑 config.yaml）
notepad config.yaml

# 6. 运行应用
python main.py
```

## 🐛 常见问题

### Q1: "ModuleNotFoundError: No module named 'PyQt6'"

**解决**：依赖未安装，运行以下命令：
```bash
pip install -r requirements.txt
```

### Q2: "API Key 无效"

**解决**：
1. 检查 `config.yaml` 中的 API Key 是否正确
2. 确认 API Key 未过期
3. 确认阿里云账号有足够余额

### Q3: 应用启动后闪退

**解决**：
1. 查看命令行是否有错误输出
2. 检查 Python 版本是否 >= 3.8
3. 尝试以管理员身份运行

### Q4: 界面显示模糊（高分屏）

**解决**：
应用已内置 DPI 感知，如仍有问题：
1. 右键 `main.py` 或生成的 `.exe`
2. 属性 → 兼容性 → 更改高 DPI 设置
3. 勾选"替代高 DPI 缩放行为"
